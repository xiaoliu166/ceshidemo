## 菜单结构（最终版）

### 左侧导航
采用 Snowflake 风格的深色侧边栏设计：

1. **首页** - 快捷入口 + 工作流程 + 最近项目 + 统计数据
2. **竞品** - 竞品分析的完整流程

   - 新建竞品分析
   - 查看分析报告（4个维度）
3. **Listing** - Listing 生成的完整流程

   - 查看生成结果
4. **资源** - 资源管理
   - 文案模板
   - 表达库
   - 提示词（竞品分析 + Listing）
   - 案例库
5. **设置** - 系统设置

## 视觉设计

## 页面功能详解

### 1. 首页（Dashboard）
**快捷入口**：
- 新建竞品分析（蓝色渐变卡片）
- 新建 Listing（紫色渐变卡片）

**工作流程**：
- 4步流程展示
- 每步带图标和渐变色

**最近项目**：
- 3个项目卡片
- 状态标签（已完成/分析中/草稿）
- 点击跳转到对应页面

**统计数据**：
- 总项目数、已完成、分析中、本月生成
- 渐变图标

### 2. 竞品页面
**三个视图**：

#### 列表视图
- 显示所有竞品分析历史
- 新建竞品分析按钮（右上角）
- 点击已完成项目查看报告

#### 新建视图
- 输入表单：
  - 产品关键词（必填）
  - 目标市场（必填）
  - 产品类目
  - 补充信息
- 分析说明卡片
- 开始分析按钮

#### 报告视图
- 4个Tab：
  1. **卖点分类**
     - 按产品属性（7类）
     - 按需求层次（马斯洛7层 + 衍生需求）
     - 按KANO模型（5类）
  2. **卖点表达**
     - FABE法则（4个维度）
     - SUCCESs模型（6个要素）
  3. **卖点关系**
     - 先后顺序
     - 主次布局
     - 表达强弱（±10分制）
  4. **评论内容**
     - 出入点
     - 情绪点
     - 反差点
     - 跳跃点

### 3. Listing 页面
**三个视图**：

#### 列表视图
- 显示所有 Listing 历史
- 新建 Listing 按钮（右上角）
- 点击已完成项目查看结果

#### 新建视图
- 选择竞品分析任务
- 生成内容选项（6项）：
  - 主图
  - 标题
  - 五点描述
  - 品牌故事
  - 详情页
  - 图片需求
- 生成 Listing 按钮

#### 结果视图
- 主图建议
- 标题（含合规评分）
- 五点描述
- 品牌故事
- 详情页图片需求
- 导出按钮

### 4. 资源页面
**4个Tab**：

#### 文案模板
- 模板卡片列表
- 评分、使用次数
- 使用模板按钮

#### 表达库
- 表达方式列表
- 示例、适用卖点
- 查看详情按钮

#### 提示词
- 竞品分析提示词列表
  - 显示名称、使用次数、最后修改时间
  - 默认标签
  - 编辑/删除按钮
  - 查看完整内容
- Listing 提示词列表
  - 同上结构

#### 案例库
- 成功案例列表
- 优化前后对比
- 下载完整案例

## 核心改进

### 1. 菜单简化
- 从复杂的多层菜单简化为5个主菜单
- 每个菜单对应一个核心功能
- 符合用户心智模型

### 2. 流程整合
- 竞品分析：列表 → 新建 → 报告（一个页面内完成）
- Listing生成：列表 → 新建 → 结果（一个页面内完成）
- 减少页面跳转，提升体验

### 3. 视觉统一
- Snowflake 风格的深色侧边栏
- 一致的卡片设计和悬停效果
- 统一的渐变色使用
- 专业的配色方案

### 4. 功能完整
- 竞品分析包含完整的4个维度
- Listing生成包含6个输出内容
- 提示词管理整合到资源页面
- 所有功能都有对应的Mock数据
