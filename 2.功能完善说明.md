## 🎯 完整用户旅程

### 旅程1：快速查看已完成项目
```
登录 → 工作台 → 点击"咖啡研磨机" 
→ 项目详情页（基本信息Tab）
→ 切换到"卖点矩阵"Tab → 查看卖点和证据
→ 切换到"Listing"Tab → 查看文案
→ 切换到"图需"Tab → 查看图片方案
```

### 旅程2：创建新项目（完整流程）
```
登录 → 工作台 → 新建项目
→ Step 1：填写产品信息
→ Step 2：添加竞品ASIN
→ Step 3：确认启动
→ 分析进度页（自动）
→ 项目详情页（自动跳转到卖点矩阵Tab）
→ 查看分析结果
→ 切换到Listing Tab生成文案
→ 切换到图需Tab生成图片方案
```

### 旅程3：管理竞品
```
登录 → 工作台 → 点击项目
→ 项目详情页 → 竞品管理Tab
→ 添加新竞品
→ 重新抓取数据
→ 删除不需要的竞品
```

### 旅程4：浏览资源库
```
登录 → 侧边栏"资源库"
→ 文案模板Tab → 查看模板
→ 表达库Tab → 查看表达方式
→ 案例库Tab → 查看成功案例
```

### 旅程5：管理账户
```
登录 → 侧边栏"设置"
→ 账户信息Tab → 修改个人信息
→ 团队管理Tab → 邀请成员
→ 通知设置Tab → 调整通知偏好
→ 订阅与账单Tab → 查看使用情况
```
