## 🚀 如何使用

### 本地运行
```bash
cd listing-ai-demo
npm install
npm run dev
```
访问：http://localhost:3000

### 构建生产版本
```bash
npm run build
npm start
```

### 部署到Vercel（可选）
```bash
# 安装Vercel CLI
npm i -g vercel

# 部署
vercel
```

## 🔧 技术说明

### 技术栈
- **Next.js 14**：React框架，App Router
- **TypeScript**：类型安全
- **Tailwind CSS**：样式框架
- **Shadcn UI**：UI组件库
- **Zustand**：状态管理
- **Lucide React**：图标库

### 特点
- ✅ 纯前端实现，无需后端
- ✅ 所有数据hardcode在前端
- ✅ 使用setTimeout模拟异步操作
- ✅ 响应式设计，支持桌面端
- ✅ 构建成功，无TypeScript错误
