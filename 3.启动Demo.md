# 🚀 AI Listing OS Demo - 启动指南

## 快速启动（3步）

### 1. 进入项目目录
```bash
cd listing-ai-demo
```

### 2. 安装依赖（首次运行）
```bash
npm install
```

### 3. 启动开发服务器
```bash
npm run dev
```

### 4. 打开浏览器
访问：http://localhost:3000

---

## 🎬 演示流程

### 完整演示路径（约5分钟）：

1. **登录页** → 点击"立即登录"
2. **工作台** → 点击"咖啡研磨机"项目（已完成）
3. **卖点矩阵** → 点击任意卖点查看证据 → 点击"生成Listing"
4. **Listing生成** → 查看文案 → 点击"生成图需方案"
5. **图需生成** → 查看图片需求

### 或者体验完整创建流程：

1. **登录页** → 点击"立即登录"
2. **工作台** → 点击"新建项目"
3. **向导Step 1** → 填写信息 → 下一步
4. **向导Step 2** → 查看竞品 → 下一步
5. **向导Step 3** → 点击"启动深度分析"
6. **分析进度** → 等待自动完成（约15秒）
7. **卖点矩阵** → 查看分析结果
8. **后续流程** → 同上

---

## ✨ 核心亮点

### 1. 卖点矩阵（最重要）
- 展示卡诺模型分类（基本型/期望型/魅力型）
- 点击卖点查看真实评论证据
- 这是产品的核心差异化功能

### 2. Listing生成
- AI生成的标题和五点描述
- 合规检查（95分）
- 关键词覆盖分析

### 3. 图需生成
- 4张图的详细需求说明
- 可直接交付设计师
- 包含目标、文案、构图建议

## 🛠️ 故障排除

### 如果端口被占用：
```bash
# 使用其他端口
npm run dev -- -p 3001
```

### 如果依赖安装失败：
```bash
# 清除缓存重新安装
rm -rf node_modules package-lock.json
npm install
```

### 如果页面空白：
1. 检查浏览器控制台是否有错误
2. 确保Node.js版本 >= 18
3. 尝试清除浏览器缓存

---

## 📞 技术支持

如遇问题，请检查：
- Node.js版本：`node -v`（需要 >= 18）
- npm版本：`npm -v`
- 端口3000是否被占用
